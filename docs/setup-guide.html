Installation and Setup

This guide will walk you through the process of setting up and running the Chlann Claude project locally.

Prerequisites

Make sure you have the following installed on your system:

  • Node.js (v14 or newer)
  • npm or yarn
  • Docker and Docker Compose (for containerized setup)
  • Git

Cloning the Repository

git clone https://github.com/m4cd4r4/chlannClaude1.git
cd chlannClaude1

Backend Setup

Navigate to the backend directory and install dependencies:

cd backend
npm install

Create a .env file based on the example:

cp .env.example .env

Update the .env file with your configuration settings.

Start the backend server:

npm run dev

Note: The backend server will run on http://localhost:3000 by default.

Mobile App Setup

Navigate to the mobile directory and install dependencies:

cd ../mobile
npm install

Start the development server:

npm start

Note: To run the app on a physical device, scan the QR code with the Expo Go app.

Docker Setup (Alternative)

For a containerized setup using Docker:

docker-compose up -d

This will start both the backend and a database container.

Troubleshooting: If you encounter any issues during setup, please check the logs or open an issue on our GitHub repository.
